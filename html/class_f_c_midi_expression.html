Detailed Description

Encapsulates the continous MIDI data in a cell.

The MIDI expression can be any of these MIDI events:

  • Controller
  • Patch change
  • Channel pressure/aftertouch
  • Pitch wheel

◆ GetPatchChangeMode()

twobyte FCMidiExpression::GetPatchChangeMode ( ) const
inline

Returns the patch change mode for a patch change event, wich defines how the patch change is sent.

Lua-supported (also as property),

Returns
Values 0-4. 0 - Standard patch change (all bank data should be 0). 1 - Bank sent as controler 0, 32; then program change (bank data 1 and 2 are used). 2 - Bank sent as controller 0; then program change (bank data 1 is used). 3 - bank sent as controller 32; then program change (bank data 2 used). 4 - 2 program changes (bank data 1 and 2 are used).
