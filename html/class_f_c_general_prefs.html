Detailed Description

The "Program Options" (program-wide options that is not stored in the document).

Please note that these options are global. Make sure to notify the user if a options is changed.

The Reflow Measures settings in the Finale Preferences needs a special combination of values, as outlined below:

Do Not Reflow:

  • GetRecalcMeasures() = false
  • GetRespaceMeasureLayout() = false
  • GetRetainFrozenMeasures() = true

Only Within Systems:

  • GetRecalcMeasures() = false
  • GetRespaceMeasureLayout() = true
  • GetRetainFrozenMeasures() = true

Across Systems (Maintain System Locks):

  • GetRecalcMeasures() = true
  • GetRespaceMeasureLayout() = false
  • GetRetainFrozenMeasures() = true

Across Systems (Remove System Locks):

  • GetRecalcMeasures() = true
  • GetRespaceMeasureLayout() = false
  • GetRetainFrozenMeasures() = false
