Detailed Description

Class for document-specific chord settings. Some additional chord-related settings are available in FCMiscDocPrefs.

Member Enumeration Documentation

◆ CHORDSTYLES

enum FCChordPrefs::CHORDSTYLES

Chord styles for use with the FCChordPrefs::GetStyle() and FCChordPrefs::SetStyle() methods.

Lua-supported.

Enumerator
CHRDSTYLE_NORMAL 

Chord roots as letters

CHRDSTYLE_ROMAN 

Chord roots as roman numerals.

CHRDSTYLE_NASHVILLE_A 

"Nashville A": Chord roots as scale degrees. Accidentals preceed root.

CHRDSTYLE_GERMAN 

Chord root in German style. (H for B. B for B flat.)

CHRDSTYLE_SOLFEGGIO 

Display roots as Solfeggio (Do, Re, Mi, etc.)

CHRDSTYLE_EUROPEAN 

Chord root in European style (display natural sign on B naturals).

CHRDSTYLE_SCANDINAVIAN 

Chord root in Scandinavian style (H for B. Force accidental on B naturals.)

CHRDSTYLE_NASHVILLE_B 

"Nashville A": Chord roots as scale degrees. Accidentals follow root.
