Detailed Description

Class for staff (or staff style) name positioning data.

To get the the default positioning for staff names, use the FCStaffNamePositionPrefs class.

This class is handled through the FCStaff class directly (FCStaff::GetFullNamePosition and FCStaff::GetAbbreviatedNamePosition).
