Detailed Description

The 4 (short, medium, long, tie ends) tie contour definitions.

Starting with Finale 26.2, it is no longer possible to Load or Save this class directly. Instead use FCTiePrefs::CreateTieContourPrefs and FCTiePrefs::SaveTieContourPrefs to load and save this class. These newer functions work on any suppoted Finale version.
