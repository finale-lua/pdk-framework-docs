◆ GetChordLayerData()

FCLayerPlaybackData * FCInstrumentPlaybackData::GetChordLayerData ( )

Returns the chord playback layer object (of the FCLayerPlaybackData class).

Please note that data obtained by this method must still be saved through the FCInstrumentPlaybackData class object, since the returned object for this method is just a sub record.

The value returned is no longer valid after the FCInstrumentPlaybackData class is destroyed. Accessing it then may cause Finale to crash.

Lua-supported.

Returns
