enum FCLyricsPrefs::SMARTHYPHEN_SYSTEMSTART

Smart hyphen values to be used with GetHyphenSystemStart and SetHyphenSystemStart.

Lua-supported.

Enumerator
SMARTHYPHEN_SYSSTART_ALWAYS 

Always put hyphen on next system.

SMARTHYPHEN_SYSSTART_ONLYLATERSYLLABLE 

Only put hyphens on next system when the first syllable is attached to the second note or later.

SMARTHYPHEN_SYSSTART_NEVER 

Never put hyphen on next system.
