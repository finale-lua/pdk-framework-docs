Detailed Description

The preferences for ties (includes options both from tie options and tie contour options).

One of the tie settings is also available in the FCMusicSpacingPrefs class as FCMusicSpacingPrefs::GetTiesGutter, since it affects the beat chart. However, as of version 0.58 it is recommended to use FCTiePrefs::GetExtraSystemStartSpace instead.
