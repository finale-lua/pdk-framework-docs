Detailed Description

The preferences for the default tuplet definition.

Please note that quite a few of these preferences does not work on versions lower than Finale 2012.

This struct requires both PDK_FRAMEWORK_PREFS and PDK_FRAMEWORK_ENTRIES to be defined.
