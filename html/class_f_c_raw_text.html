◆ GetSubString()

bool FCRawText::GetSubString ( FCString * pString,
const FCRange * pRange ) const

Gets a string object with the range of characters from loaded text.

This function eliminates the overhead of calling CreateString followed by FCString::MakeSubString.

Warning
FCRawText represents characters internally as UTF-16-encoded code points. If the string includes characters outside the Basic Multilingual Plane (BMP), this function could potentially return a partial character. Lua clients should use particular care with this function, because converting from indices on UTF-8 strings to indices on UTF-16 strings is not straightforward. A much safer approach for Lua would be to load CreateString into a Lua string and then use the built-in utf8 Lua library to process the string.

Lua-supported (0.68).

Parameters
pStringthe string to which to copy the substring.
pRangethe range of characters to copy.
Returns
Returns false if the range is invalid.
