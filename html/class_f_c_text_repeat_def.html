Detailed Description

The class for a text repeat definition.

Please note that when deleting objects of this type, make sure to use DeepDeleteData(). Also, make sure to use DeepSaveAs() rather than SaveAs().

For convenience, this class also have a DeepSaveNew() method that's easier to use than SaveNew().
