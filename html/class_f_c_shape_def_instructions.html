◆ AddCurveTo()

FCShapeDefInstruction * FCShapeDefInstructions::AddCurveTo ( int  ctrlpt_height = 72)
inline

Adds a standard CurveTo instruction.

NOTE: Since the Shape Designer seems to define all curveto instructions in an almost identical way and control the size and orientation of the curve through the StartObject instruction, this method contains no parameters except the height (the Shape Designer defaults are filled in).

Lua-supported.

Parameters
ctrlpt_heightThe bezier control point's y value, where 72 is the default in this case. A negative value would result in an inverted curve.

◆ AddCurveToDetails()

FCShapeDefInstruction * FCShapeDefInstructions::AddCurveToDetails ( int  x1,
int  y1,
int  x2,
int  y2,
int  endpt_x,
int  endpt_y 
)
inline

Adds a CurveTo instruction with all parameters set.

Here are the 90 degrees clockwise movements (at standard orientation):

  • Curve in bottom left corner: -72, 0, -72, 72, 0, 72
  • Curve in top left corner: 0, 72, 72, 72, 72, 0
  • Curve in top right corner: 72, 0, 72, -72, 0, -72
  • Curve in bottom right corner: 0, -72, -72, -72, -72, 0

Here are the 90 degrees counter-clockwise movements (at standard orientation):

  • Curve in bottom left corner: 0, -72, 72, -72, 72, 0
  • Curve in top left corner: -72, 0, -72, -72, 0, -72
  • Curve in top right corner: 0, 72, -72, 72, -72, 0
  • Curve in bottom right corner: 72, 0, 72, 72, 0, 72

Lua-supported.

Parameters
x1x coord for ctrl outer control point
y1y coord for ctrl outer control point
x2x coord for destination point
y2y coord for destination point
endpt_xx coord for inner control point
endpt_yy coord for inner control point

inline

Adds a StartObject instruction.

This instruction should always start an object in a shape.

Lua-supported.

Parameters
pOriginThe x, y coordinate origin where the drawing starts.
pBoundLeftTopThe left, top boundary coordinate of the object.
pBoundRightBottomThe right bottom boundary coordinate of the object.
xtransformThe horizontal transformation factor * 1000. Should be 1000 for an untransformed object.
ytransformThe vertical transformation factor * 1000. Should be 1000 for an untransformed object.
rotationThe rotation transform value. (It seems like 90 degrees equals 1024.)
Returns
NULL if any of the point values are NULL. Otherwise, pointer to the created object.
