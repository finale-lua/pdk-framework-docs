Detailed Description

Class for a multi-measure rest instance.

The info is stored in the Finale database at the start measure position, so loading a MM rest is done with a Load(measure_number) call.

SaveNew() is disabled for this method. Instead Save() needs to be called, even when creating new multimeasure rests.
