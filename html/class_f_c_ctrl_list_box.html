Detailed Description

A list box control.

The contents of the list box control can be set with AddString or SetStrings.

On Windows, it's a native list box control. On Mac/Cocoa, it's a NSTableView control WITH NO PREDEFINED COLUMNS that emulates a list box behaviour.

IMPORTANT! If resizing of the window should be used on Windows, make sure that the LBS_NOINTEGRALHEIGHT attribute is set for the list box resource.

The class inheritance is different on Windows (inherits from FCControl) compared to the Mac (inherits from FCCtrlDataList)

Programmatically created controls of this class defaults to a width of 150 points and a height of 170 points.
