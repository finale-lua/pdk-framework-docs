Detailed Description

The class for a smart shape entry connection style.

This is a special class that doesn't save its own document data. Instead, the data is stored by the FCSmartShapePrefs class. FCSmartShapeEntryConnectStyle is not useful in versions of Finale before Finale 26.2, and it always returns zero for all values in unsupported Finale versions.

To an instance of this class, call FCSmartShapePrefs::CreateEntryConnectStyle.

This class was added in version 0.65.
