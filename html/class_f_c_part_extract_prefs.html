Detailed Description

Preferences that controls the part extraction options in the document.

Despite the mention of Lua-supported properties, this class is not hooked up to Lua. It appears to be obsolete in the post Finale 2007 era of linked parts.

Member Enumeration Documentation

◆ EXTRACT_SPACING

enum FCPartExtractPrefs::EXTRACT_SPACING

The constants to use for controlling the spacing of the music for the extracted part.

Enumerator
EXTRACT_SPACING_TIMESIG 

Apply Time Sig Spacing on the extracted part.

EXTRACT_SPACING_BEAT 

Apply Beat Spacing on the extracted part.

EXTRACT_SPACING_NOTE 

Apply Note Spacing on the extracted part.

EXTRACT_SPACING_PRESERVECURRENT 

Preserve current spacing.
