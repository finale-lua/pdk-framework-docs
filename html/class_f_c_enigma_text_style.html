Detailed Description

Class that specifies Enigma text style. This includes font, size, efx, superscript, baseline adjustments, and interspacing. MakeMusic has provided a mechansism whereby it can include color is well, but as of at least Finale 27.3 there is no support for color in Enigma strings.
