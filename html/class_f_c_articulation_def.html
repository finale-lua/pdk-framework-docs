Detailed Description

The class for an articulation definition. On Finale 2012 and above, this class supports the Unicode character range.

Member Enumeration Documentation

◆ ARTPOSITIONING_SIDES

enum FCArticulationDef::ARTPOSITIONING_SIDES

The auto positioning modes for articulations. Used by FCArticulationDef::GetAutoPosSide() and FCArticulationDef::SetAutoPosSide()

Any of these values:

  • ARTPOS_MANUAL_POSITIONING - Manual positioning
  • ARTPOS_AUTO_NOTESTEM - Auto note/stem side
  • ARTPOS_ALWAYS_NOTE_SIDE - Always on note side
  • ARTPOS_ALWAYS_STEM_SIDE - Always on stem side
  • ARTPOS_BELOW_ENTRY - Always below the entry
  • ARTPOS_ABOVE_ENTRY - Always above the entry
  • ARTPOS_ALWAYS_ON_STEM - Always on stem (Finale 26+

Lua-supported. (ARTPOS_ALWAYS_ON_STEM added 0.58)

◆ ARTSLUR_INTERACTION

enum FCArticulationDef::ARTSLUR_INTERACTION

The options for slur interaction used by FCArticulationDef::GetSlurInteraction() and FCArticulationDef::SetSlurInteraction()

Any of these values:

  • ARTSLURINT_IGNORE - Ignore slurs
  • ARTSLURINT_INSIDE - Inside slurs
  • ARTSLURINT_AUTO_INOUT - Auto Inside/Outside

Finale 26 and higher

Lua-supported (0.58)
