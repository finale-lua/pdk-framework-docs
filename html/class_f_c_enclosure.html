Constructor & Destructor Documentation

◆ FCEnclosure()

FCEnclosure::FCEnclosure ( )
inline

The constructor.

Some Finale classes embed the enclosure (specifically FCMeasureNumberRegion and FCSeparateMeasureNumber). For these, use the GetEnclosure... methods and do not use any constructor. The enclosure(s) will be saved when you save the parent class.

Otherwise, you can use this constructor to create a new enclosure and assign it as follows:

FCTextExpressionDef: Call SaveAs on the constructed FCEnclosure instance, using the same item number as the instance of FCTextExpressionDef.

FCTextRepeatDef: Use this contructor but call FCTextRepeatDef::SaveEnclosure to save it.

For both, you should first check to see if there is an existing enclosure using either FCTextExpressionDef::CreateEnclosure or FCTextRepeatDef::CreateEnclosure. If the enclosure already exists, you do not need to use this constructor.

Lua-supported (0.60).
